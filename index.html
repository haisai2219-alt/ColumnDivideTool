ColumnDivideTool
機能 使い方 ダウンロード

CSVを、カラムで分ける。

設定ファイルを1行書いて、batをダブルクリック。
指定したカラムの値ごとにCSVを自動分割します。

無料ダウンロード（Windows / Java 11+）

v1.0.0 — 無料・広告なし

📂

複数ファイルを一括処理

inputフォルダに置いたCSVをまとめて処理。ファイルをひとつひとつ開く必要はありません。

✂️

行数上限で自動分割

1ファイルの最大行数を指定すると、超えた分は自動で連番ファイルに分割されます。

🔍

完全一致・部分一致フィルタ

特定の値だけ抽出して分割することも可能。不要なデータを除外した出力が作れます。

📝

テキスト設定ファイル

config.txt を編集するだけで全設定が完了。GUIなしでも直感的に使えます。

使い方

1

ZIPをダウンロードして解凍する

ColumnDivideTool フォルダをデスクトップなど任意の場所に置く

2

input/ に分割したいCSVを置く

複数ファイルをまとめて入れてもOK（UTF-8形式）

3

config.txt を編集する

targetColumnName に分割基準のカラム名を書くだけでOK

4

columnDivide.bat をダブルクリック

処理完了後、output/ フォルダに分割済みCSVが出力される
